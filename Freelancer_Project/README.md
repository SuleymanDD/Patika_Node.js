# Freelancer Portfolyo Sitesi (Dinamik Backend)

Bu proje, StartBootstrap'in **Freelancer** teması kullanılarak hazırlanmış statik bir web sitesinin, **Node.js** ve **Express.js** teknolojileriyle dinamik bir web uygulamasına dönüştürülmesini kapsamaktadır. Proje kapsamında, statik içerikler veritabanına taşınmış ve yönetilebilir hale getirilmiştir.

---

## 🛠 Teknoloji Yığını
* **Node.js**
* **Express.js**
* **MongoDB (Mongoose)**
* **EJS (Template Engine)**
* **HTML/CSS (Bootstrap)**

---

## ✨ Özellikler
* **Dinamik İçerik Yönetimi:** Portfolyo içeriklerinin veritabanından çekilmesi.
* **CRUD İşlemleri:** Proje içeriklerinin eklenmesi, düzenlenmesi, silinmesi ve listelenmesi.
* **Veritabanı Entegrasyonu:** MongoDB ile verilerin kalıcı hale getirilmesi.

---

## 🚀 Kurulum Adımları

```bash
1. Depoyu klonlayın:
git clone https://github.com/SuleymanDD/Patika_Node.js.git

2. Projenin bulunduğu klasöre gidin:
cd Patika_Node.js/Freelancer_Project

3. Gerekli bağımlılıkları yükleyin:
npm install

4. .env dosyasını oluşturun ve veritabanı bağlantı bilgilerini ekleyin:
PORT=3000
MONGODB_URI=mongodb_baglanti_adresiniz

5. Uygulamayı başlatın:
npm run start
```
